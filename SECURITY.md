# Security Policy

## Supported versions

Security fixes are provided for the current major version only.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a vulnerability

Please report security vulnerabilities privately through GitHub's [private vulnerability reporting](https://github.com/coffee-and-fun/konami-code-detector/security/advisories/new) feature.

Please do not open a public issue, pull request, or discussion for security reports. You should receive an acknowledgement within a few business days. Once a fix is available, a patch release and a GitHub Security Advisory will be published.
